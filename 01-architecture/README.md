# 01-architecture

The target Azure architecture for the distributed Splunk Enterprise Security + UBA lab: the mermaid topology diagram, the IP/subnet plan and CIDR rationale, component-to-role mapping, planned NSG rules, and the course gaps the architecture exposes (clustering, SHC, MC, ES admin, UBA). This is the reference every concept and lab ties back to — when a feature is explained, it gets anchored here so infrastructure stays concrete rather than abstract.
