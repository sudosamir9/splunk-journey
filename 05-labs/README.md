# 05-labs

The hands-on record of the Azure Splunk lab, split four ways. `runbooks/` holds reusable "how to do X" procedures. `configs/` holds sanitized Splunk `.conf` files. `journal/` holds dated entries documenting what I built, what broke, and how I fixed it — the most valuable artifacts in this repo and portfolio gold, treated as deliverables rather than scratch. `azure-setup/` holds the `az` CLI command log, network design, NSG rules, and VM specs (semi-IaC by hand, making the eventual Terraform leap trivial).
