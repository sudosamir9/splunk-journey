# 00-meta

Project metadata and the connective tissue that keeps the journey navigable across variable, multi-day sessions: the decisions log, chat/setup summaries, the phase-based roadmap, the NotebookLM workflow, the Obsidian vault config, and note templates. Archived `handoff.md` snapshots live in `handoffs/` (the dated learning timeline) and Templater note templates live in `templates/`. Start here — alongside the root `handoff.md` — when you need to remember where you are and why decisions were made the way they were.
