# 99-notebooklm

NotebookLM source bundles, one folder per topic (`<topic-slug>/`), each assembled as a production input — not just a review afterthought. A bundle gathers the relevant transcripts, lecture notes, topic synthesis, concept atoms, lab journal entries, and runbooks, plus a `glossary.md`, `quiz-seeds.md`, and `prompts.md`, ready for upload. From each bundle I generate four outputs: podcast, quizzes, mind map, and briefing doc. Downloaded outputs land in a gitignored `exports/` subfolder. Full conventions in `00-meta/notebooklm-workflow.md`.
